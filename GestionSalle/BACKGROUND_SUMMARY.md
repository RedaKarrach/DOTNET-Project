### Personnalisation Avancée

   - Remplacer dans `home.css`

   - Configurer les paramètres
   - Superposer au background

   - Ajouter scrollmagic.js
   - Animer le background au scroll
   - Créer profondeur 3D

4. **Thème jour/nuit**
   - Créer variante claire
   - Toggle CSS avec JavaScript
   - Sauvegarder préférence user

## ?? Support

### Problème de Compilation?
1. Vérifier la syntaxe CSS
2. Fermer et rouvrir Visual Studio
3. Clean + Rebuild solution
4. Vérifier les accolades `{}`

### Background pas visible?
1. Vider le cache (Ctrl+F5)
2. Vérifier fichier `home.css` chargé
3. Inspecter élément (F12)
4. Regarder console erreurs

### Texte illisible?
1. Ajuster couleurs du dégradé
2. Utiliser fond plus sombre
3. Augmenter contraste
4. Tester Option 1, 5 ou 7
