## ?? EFFETS SUPPLÉMENTAIRES

### Grille Moderne (Recommandé)

À ajouter **APRÈS** `.hero-section` dans le CSS:

```css
.hero-section::before {
    content: '';
 position: absolute;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    background-image: 
        linear-gradient(rgba(255, 255, 255, 0.03) 1px, transparent 1px),
        linear-gradient(90deg, rgba(255, 255, 255, 0.03) 1px, transparent 1px);
    background-size: 50px 50px;
 opacity: 0.5;
    pointer-events: none;
}
```

### Lumière Rotative (Animation douce)

À ajouter **APRÈS** `.hero-section::before`:

```css
.hero-section::after {
    content: '';
    position: absolute;
    top: -50%;
    right: -50%;
    width: 200%;
    height: 200%;
    background: radial-gradient(circle, rgba(255, 107, 53, 0.1) 0%, transparent 70%);
    animation: rotateGlow 20s linear infinite;
    pointer-events: none;
}

@keyframes rotateGlow {
    0% {
        transform: rotate(0deg);
    }
    100% {
        transform: rotate(360deg);
    }
}
```

### Particules Flottantes (Effet neige)

```css
.hero-section::before {
    content: '';
    position: absolute;
    top: 0;
    left: 0;
    right: 0;
    bottom: 0;
    background-image: 
  radial-gradient(circle, rgba(255, 255, 255, 0.5) 1px, transparent 1px);
    background-size: 50px 50px;
    animation: particleFloat 15s linear infinite;
}

@keyframes particleFloat {
    0% {
      background-position: 0 0;
    }
    100% {
        background-position: 50px 50px;
    }
}
```

## ?? PERSONNALISATION AVANCÉE

### Modifier l'Angle

```css
/* Diagonal standard (recommandé) */
background: linear-gradient(135deg, ...);

/* Horizontal gauche ? droite */
background: linear-gradient(90deg, ...);

/* Vertical haut ? bas */
background: linear-gradient(180deg, ...);

/* Diagonal inversé */
background: linear-gradient(-45deg, ...);
```

### Ajouter des Couleurs

```css
/* 2 couleurs */
background: linear-gradient(135deg, #color1 0%, #color2 100%);

/* 3 couleurs (recommandé) */
background: linear-gradient(135deg, #color1 0%, #color2 50%, #color3 100%);

/* 4 couleurs */
background: linear-gradient(135deg, #color1 0%, #color2 33%, #color3 66%, #color4 100%);

/* 5 couleurs */
background: linear-gradient(135deg, #color1 0%, #color2 25%, #color3 50%, #color4 75%, #color5 100%);
```

### Créer un Dégradé Radial

```css
/* Depuis le centre */
background: radial-gradient(circle, #color1 0%, #color2 100%);

/* Depuis le coin haut-gauche */
background: radial-gradient(circle at top left, #color1 0%, #color2 100%);

/* Ellipse horizontal */
background: radial-gradient(ellipse, #color1 0%, #color2 100%);
```
